# Telegram Bot Module

Модуль для работы с Telegram ботом, использующий aiogram и LLM модель для генерации ответов.

## Возможности

- Отправка неограниченного количества сообщений
- Интеграция с LLM моделью для генерации ответов
- Обработка команд `/start`, `/clean_history`, `/cancel`
- Автоматическое разбиение длинных сообщений
- Управление сессиями пользователей
- Логирование всех операций

## Установка

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Заполните переменные окружения в файле `.env`:
   - `TELEGRAM_BOT_TOKEN` - токен вашего бота от @BotFather
   - `LLM_BASE_URL` - URL API для подключения к модели

## Запуск

```bash
python tg-module/bot.py
```

## Структура модуля

- `bot.py` - основной модуль бота с обработчиками команд
- `config.py` - конфигурация и настройки
- `llm_client.py` - клиент для работы с LLM API
- `requirements.txt` - зависимости Python
- `.env.example` - пример файла с переменными окружения

## API LLM

Бот работает с API из папки `model`, который должен быть запущен отдельно. API поддерживает следующий эндпоинт:

- `POST /start_talk/` - генерация ответа на сообщение пользователя

Формат запроса:

```json
{
    "user_id": "123456789",
    "prompt": "текст сообщения пользователя",
    "parameters": {}
}
```

Формат ответа:

```json
{
    "user_id": "123456789",
    "msg": "ответ от LLM модели",
    "timestamp": "2024-01-01 12:00:00"
}
```

## Команды бота

- `/start` - начать работу с ботом
- `/help` - показать справку
- `/clean_history` - очистить историю общения

## Логирование

Бот ведет подробные логи всех операций, включая:

- Запуск и остановку бота
- Обработку команд пользователей
- Ошибки при работе с LLM API
- Статистику сообщений пользователей
