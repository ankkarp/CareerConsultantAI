### 6. Observability / Evals (Наблюдаемость)

**Модули:** `tg-module`, `monitoring-stack` (папка существует, но содержимое не раскрыто)

| Категория                 | Метрики и логи                                                                                                                                                                                                                                                                                                                                                               |
| ---------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Метрики**           | Подразумеваются: количество обработанных сообщений, количество активных сессий, задержки ответа LLM, успешность/ошибки вызовов API.`monitoring-stack` вероятно, содержит конфигурацию для Prometheus.                                        |
| **Логи**                 | `tg-module/bot.py` логирует: запуск/остановку бота, все команды пользователей, ошибки LLM API, статистику сообщений. Формат — стандартный вывод (stdout), может собираться в ELK/Loki через `monitoring-stack`.                                               |
| **Трейсы**             | Могут быть реализованы через стандартные middleware FastAPI (для `model`) и aiogram (для бота) с интеграцией в Jaeger/Zipkin.                                                                                                                                                                                              |
| **Проверки (Evals)** | Для оценки качества поиска профессий можно использовать метрики ранжирования (MRR, Recall@K) на тестовых запросах. Для оценки ответов агента — пользовательская обратная связь или авто-эвалюация с LLM-асистентом. |
