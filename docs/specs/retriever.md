### Retriever (поиск по профессиям)

**Модуль:** `vector_index`

| Параметр                 | Спецификация                                                                                                                                                                                                                                                        |
| :------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Источники**     | агрегированные данные о профессиях с hh.ru в формате json                                                                                                                                                                               |
| **Индекс**           | FAISS (Facebook AI Similarity Search). Строится скриптом `build_faiss_index.py`. Артефакты: `index.faiss` (бинарный индекс), `index.pkl` (метаданные), `docs.json` (карта документов).                  |
| **Эмбеддинги**   | YandexGPT Embeddings через API (`yandex_embeddings.py`). Ограничения: батчи по 5 документов с задержкой ~1.2 сек для соблюдения квот API.                                                                      |
| **Поиск**             | kNN (k-ближайших соседей) семантический поиск по текстовому запросу. CLI пример:`search_professions.py --query "Data Scientist" --k 5`.                                                                            |
| **Ограничения** | Индекс статичен, требует полной пересборки (`build_faiss_index.py`) при изменении источника данных. Зависит от доступности внешнего API Yandex Cloud для построения. |
