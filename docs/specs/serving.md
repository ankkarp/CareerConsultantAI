### Serving / Config (Запуск и конфигурация)

**Корень проекта:** `docker-compose.yml`, `run_model_api.py`, `tg-module/bot.py`

| Параметр                      | Спецификация                                                                                                                                                                                                                                                                                            |
| ------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Запуск**                | **Модель (LLM API):** `python run_model_api.py` (запускает FastAPI/Uvicorn сервер). **Telegram бот:** `python tg-module/bot.py`. **Сервисы в сборе:** `docker-compose up` (предполагает оркестрацию всех сервисов). |
| **Конфигурация**    | Файл `.env` в корне проекта. Обязательные переменные: `TELEGRAM_BOT_TOKEN`, `LLM_BASE_URL`, `YANDEX_CLOUD_API_KEY`, `YANDEX_CLOUD_FOLDER`. Есть пример `.env.example`.                                                                                 |
| **Секреты**              | Управляются через переменные окружения, не коммитятся. В Docker-среде передаются через `docker-compose.yml` или secrets.                                                                                                                   |
| **Версии моделей** | Не специфицированы в явном виде. Зависят от эндпоинта `LLM_BASE_URL` и используемой модели YandexGPT (для эмбеддингов и, предположительно, для генерации).                                             |
