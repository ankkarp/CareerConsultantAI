
### Tools / APIs (Внешние интерфейсы)

**Модули:** `tg-module` (клиент), `model` (сервер), `web_search`

| Параметр             | Спецификация                                                                                                                                                                                                                                                                                                                    |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Контракты** | **LLM API (`model`):** `POST /start_talk/`Запрос: `{"user_id": "string", "prompt": "string", "parameters": {}}`Ответ: `{"user_id": "string", "msg": "string", "timestamp": "string"}`**Telegram Bot:** использует aiogram для взаимодействия с Telegram Bot API.  |
| **Ошибки**       | Логирование ошибок API LLM (недоступность, таймауты, невалидный ответ) в `tg-module/bot.py`. Бот продолжает работу, уведомляя пользователя о проблеме.                                                                             |
| **Timeout**            | Не указаны явно, но логика подразумевает стандартные таймауты HTTP-клиента. При разрыве соединения бот логирует ошибку.                                                                                                                    |
| **Side Effects**       | **Telegram Bot:** отправка сообщений пользователю, управление сессиями. **LLM API:** генерация ответа, потенциально изменение внутреннего состояния сессии (в памяти).                                      |
| **Защита**       | Токены и ключи (TELEGRAM_BOT_TOKEN, YANDEX_CLOUD_API_KEY) хранятся в `.env`, исключены из репозитория. Валидация входных данных через Pydantic модели не видна, но подразумевается стандартными библиотеками. |
