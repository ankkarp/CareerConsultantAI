# Model Module

Модуль для c LLM.

## Описание
Обрабатываем пост запрос из телеграма и строим персональный диалог с каждым пользователем, с сохранением текущего этапа

Построение диалога с пользователем происходит в несколько этапов
1) Узнаем основную информацию о пользователе - этап WHO
2) Классифицируем пользователя - школьник / студент / работающий - этап ABOUT
3) Собираем портрет пользователя - для каждого класса строим свой диалог - этап ABOUT
4) На основе собранной информации составляем профориентационные тесты - этап TEST
5) Проводим тесты - этап TEST
6) Формируем полный портрет пользователя для рекомендации профессии - этап RECOMMENDATION
7) Делаем рекомендацию - этап RECOMMENDATION
8) Обсуждение конкретной профессии / запрос новых рекомендаций - этап TALK

## Установка


1. Заполните переменные окружения:
   - `YANDEX_CLOUD_FOLDER` - идентификатор диалога
   - `YANDEX_CLOUD_API_KEY` - API-ключ для LLM
   - `test_run_version` - вариант проведения профориентационного теста
     - v1 - вопросы тесты генерирует LLM
     - v2 - задаются заготовленные вопросы

## Запуск

```bash
python model/app_run.py
```

## Структура модуля

- `app_run.py` - запуск сервиса с моделью
- `config.yaml` - системные промты и тулы
- `main.py` - API-приложение
- `chema.txt` - форматы для запросов
- `start_llm` - формирование ответов и создание диалога
